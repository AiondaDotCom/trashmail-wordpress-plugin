Generate unique disposable contact addresses for each visitor.

== Description ==

Add a mailto link to your site so that every visitor gets a unique disposable
address, generated through the TrashMail service.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/trashmail-contact-me` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the [tm-contact] shortcode anywhere in your site to insert a mailto link.
1. Alternatively, add a TrashMail menu item to your menu.
